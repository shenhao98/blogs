oft字体在html中无法使用

疑问

比如寒蝉活宋体_復_mianfeiziti.com.otf这个字体有大约30MB,一开始我认为文件是损坏或者过大加载不出来，又或者网站不能加载oft格式的字体文件，但是后面经过电脑安装和其它工具的预览证明文件没问题网站上也是能预览，所以至今这个问题怀疑就是文件里面有些字符有毛病导致显示不出来。

解决办法

使用python安装这个库来处理 git地址

pip install fonttools

使用

pyftsubset ./target.otf --text-file=text.txt

target.otf是需要提取的字体文件，text.txt所要提取的文字，最后会在当前目录生成target.subset.otf，enjoy! 🥳
