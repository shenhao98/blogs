Promise

const fun = () => {
  return 1
}
const promiseResult = [fun].filter(async v => v() !== 1)
console.log(promiseResult.length)  // 答案：length = 1

可能细心的你会认为，filter和我平时用法没什么区别，那么结果应该是 0，但是结果是1，这是为什么呢？

这是因为filter的回调函数返回的是一个promise，filter里面的函数每次执行返回都是真，那么结果当然是1咯！
