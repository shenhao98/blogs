## 一、音标总览

`ʃ  ʒ  tʃ  dʒ  θ  ð  ŋ  tr  dr  ts  dz`

## 二、逐个对应：音标 \+ 发音口诀 \+ 代表单词 \+ 发音要点

| 音标 | 中文谐音  | 记忆单词 | 发音要点                     |
| ---- | --------- | -------- | ---------------------------- |
| /ʃ/  | 师（轻）  | shop     | 翘舌送气，像汉字 “师”        |
| /ʒ/  | 日（轻）  | pleasure | 翘舌震动，轻声 “日”          |
| /tʃ/ | 吃        | chair    | 翘舌爆破，同 “吃”            |
| /dʒ/ | 知        | jump     | 翘舌浊音，同 “知”            |
| /θ/  | 咬舌思    | three    | 舌尖放上下牙之间，送气不震动 |
| /ð/  | 咬舌兹    | this     | 舌尖咬齿间，嗓子震动         |
| /ŋ/  | 后鼻音 ng | sing     | 舌根后抵，闭嘴后鼻音         |
| /tr/ | 戳        | tree     | 特 \+ 若快速连读             |
| /dr/ | 桌        | dress    | 德 \+ 若快速连读             |
| /ts/ | 次        | cats     | 特 \+ 斯，单词复数结尾       |
| /dz/ | 兹        | beds     | 德 \+ 兹，单词复数结尾       |

## 三、极简背诵口诀

shop 翘舌师，pleasure 轻声日；
chair 就是吃，jump 就是知；
three 咬舌思，this 咬舌兹；
sing 后鼻音，tree 戳、dress 桌；
cats 读次，beds 读兹。

## 四、补充区分：元音 \&amp; 辅音

1. **元音**：气流不受阻碍，是单词音节主干，相当于拼音韵母
   包含：长元音、短元音、双元音，共 20 个

2. **辅音**：唇 / 齿 / 舌挡住气流，辅助发音，相当于拼音声母
   你记的这 12 个，全是**辅音里最特殊、汉语拼音没有的**。
